# **App Name**: YourDropship

## Core Features:

- Product Catalog Display: Display product listings with details such as images, titles, prices, and reviews.
- Filtering and Sorting: Enable users to filter products by price range and sort products.
- Shopping Cart: Enable users to add and remove products from a shopping cart, which stores the list of selected products for checkout.
- Dropshipping Supplier Sync: Use the provided settings to ensure automated dropshipping operations via supplier list/ api integration.
- SEO Optimization: Make the shop's landing pages optimized to be discovered by the target audience by pre-filling Title and Description meta tags.
- AI product suggestion tool: Recommend similar products based on a user's viewing history and current product using AI to analyze product attributes and user preferences.
- Parallax Scrolling Effects: Incorporate parallax scrolling effects for a more engaging user experience as users scroll down the page.

## Style Guidelines:

- Primary color: Deep forest green (#228B22) to reflect nature and freshness of plants.
- Background color: Light beige (#F5F5DC) for a soft, natural feel.
- Accent color: Terracotta orange (#E2725B) for warmth and highlights.
- Body: 'PT Sans', a humanist sans-serif for a modern and slightly warm feel. Headlines: 'PT Sans' for consistency and clarity.
- Use minimalist line icons for categories and interactive elements, keeping a clean and modern aesthetic.
- Implement a responsive grid layout with clear divisions for product cards. Use full-width banners with parallax scrolling for featured items or promotional sections.
- Subtle fade-in animations when loading product images or transitioning between sections to enhance the user experience.
